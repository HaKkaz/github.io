特殊選材 心得分享 (資訊工程學系)

Author: Aann Lin Date: January 9, 2021  1:19:24

結果

先說個結果，避免有人看完才發現我是一堆備取的廢物(X)

  1. 成大資工甲組 備取3
  2. 師大資工 備取3
  3. 中央資工 備取6
  4. 清大資工 正取
  5. 交大資工 備取2
  6. 清大不分系 初試未過

(依照面試順序排序。)

My自我介紹 & My備審內容

自我介紹跟我的經歷可以點擊BLOG右上角的About看喔~

我的備審其實沒有甚麼花邊設計，就只有白底黑字這樣的word檔案，然後用紅字及粗體標示了一些自己的獎項跟關鍵字詞，我認為清交的教授應該都不會特別注重設計這方面，從初審分數跟放榜結果上應該可以反映出來，其他學校我就不確定ㄌ。

格式每間學校略有不同，但我都是複製貼上改學校名稱而已，格式大多數是 “自傳” + “申請動機” + “讀書計畫”。我的大概word總共三頁而已，因為我很爛，所以就不放上自己的備審資料了，有興趣可以私訊我的任何社交軟體要我的備審，我會斟酌給的。

備審準備方向

以下依序是我覺得備審的重要內容。

  1. 逆境向上
    這真的很重要，相信沒有人不喜歡逆境向上的人。

  2. 得獎紀錄
    這應該不用多說。

  3. 自學能力
    不一定是最重要的，但是可以展現一些自己平常學習的方法讓教授參考。

  4. 附加價值
    你能帶給這間大學甚麼? 你擁有甚麼其他特質?

  5. 成長經歷
    多少描述一下即可，我覺得這不在評分重點當中。

  6. 其它
    寫點你自己想寫的東西吧!

面試準備方向

我認為面試準備真的很重要，即使你講的話都是真的，沒有實際演練過還是會很生疏、不流暢
除了一些專業問題當場教授會給你題目以外，你一定要對你備審資料有充足的了解，當然你備審裡面的內容也都得是貨真價實的才行。

我提出一些我認為很需要準備的面試問題，以我自己為例子。

  1. 因為我高二資訊全國賽三等獎，但是高三沒得獎，所以我在去交大面試之前就已經想好要怎麼解釋自己失常了。

  2. 我的在校成績很差，是全校倒數的那種，所以這一定也會成為教授好奇的點。

  3. 如果你上了其他學校你會怎麼選擇? 這有很多種回答方向，看你要誠實還是說謊我覺得都可以準備一下XD
    (不過我很幸運的除了成大沒人問我這個問題)。

  4. 專業問題相關，以我打演算法競賽為例，他會問 : 最喜歡的算法? 覺得最困難的題目?

而我準備面試的方法是在朋友面前直接練習自我介紹，先打好一些大綱，但沒有背起來，因為我覺得就算背起來一緊張也就忘光了。

各校經驗分享

成大資工甲組

聽說成大資工很在意在校成績，成大不希望你成績太差勁，怕收你進去你念不到畢業證書，所以在校成績很差的可以不要浪費錢投成大甲組(?)

流程大概是:

  1. 自我介紹

  2. 教授問專業問題

    • 給定一序列，找出序列中的絕對眾數（保證存在）。
      (我當初說了 $O(NlgN)$ 的解，教授問我能不能再更好我就講了一些小優化，但實際上可以 $O(N)$ ，我當時沒想到。)

    • 一開始黑球白球各有五百顆在袋子中，每次拿出兩顆球不放回，如果拿出的兩顆球同色就放一顆白球回去，如果兩個不同放黑球回去，問最後剩下白球的機率。

  3. 教授問一些關於你的問題

    • 重考的決心

    • 如果比賽進展不順利怎麼辦

    • 對大學課程瞭不瞭解

    • 板中的競賽風氣

我覺得教授問的問題都挺有深度的，我很喜歡。
最後備取3其實沒有很意外，因為聽說我書審分數是12個進複試最低的，所以其實已經利用面試搬回不少成績了><

師大資工

師大資工的面試比較沒壓力，分兩間教室，各十分鐘。

流程大概是

  1. 自我介紹

  2. 教授問專業問題

    • 提出一個排序演算法

    • 指標是甚麼

  3. 教授問一些關於你的問題

    • 我當初重考的打算

    • 怎麼學習程式

  4. 教授要你問他們問題(關於師大)

    • 可以參加甚麼演算法相關的活動、出題、驗題

  5. 進第二間教室後大概是相同的流程

聽說你成績太好他會叫你去學測的APCS組(X)

中央資工

聽說中央的面試順序是依照初試成績排的，我好像排第七的樣子。

面試不到10分鐘的樣子，一間教室三個教授。

面試的時候只問我有沒有考APCS跟在校成績為甚麼不好，備審上面都有寫，讓我感覺是來混薪水或是懶得看學生備審，沒有認真選才，抱歉講的不是很好聽(X)。
(也可能我是垃圾他們拿我濫竽充數複試人數)

真的不用浪費我的通勤費跟時間，問兩個問題不到三分鐘，還要大老遠從我家公車 -> 捷運 -> 火車 -> 公車通勤到貴校。

清大資工

因為NPSC決賽跟面試撞期，所以我並沒有去清大面試而是在家用skype面試。

教授有三位，貌似跟現場面試不同，參考價值較低。

流程如下:

  1. 自我介紹

  2. 專業問題

    • 我被問直條圖上最大矩形面積。
      (我先講 $O(N^2)$ 的枚舉左右界+稀疏表，再提出單調隊列 $O(N)$ 的作法)

    • 同學被問帶權重N皇后問題
      (位元DP 、 KM算法都可)

  3. 問我NPSC相關問題

    • 覺得難度如何
    • 最喜歡的題目
    • 最後名次

  4. 未來競賽規劃

    • 全國賽有沒有進
    • TOI初選
    • 大學ICPC

交大資工

流程如下:

  1. 自我介紹

  2. 教授發問

    • 當初重考到板中的契機

    • 打比賽花很多時間的題目

    • 如果遇到不會的問題會怎麼處理

    • 有沒有做過專案

    • 全國賽怎麼退步了

    • 還會不會打下學期資奧初選

  3. 專業問題

    問test()回傳值。

各校取才方向

  1. 清華大學 : 演算法
  2. 交通大學 : 演算法、資安
  3. 中央大學 : 我不在乎也不清楚
  4. 師範大學 : 演算法、(成績不要太好)
  5. 成功大學 : 資安、科展、演算法、專案 + 成功大學績要好一點

以上僅供參考，如果你有過人之處我相信無論大學選才方向如何都會看上你的!

給學弟妹的建議

如果你對資訊領域有熱忱，只要願意把時間投入資訊相關的領域，並且認真學習、努力，肯定會有成果的。
我是一個天分並不高的人，沒讀過數理資優班甚至考不上，雖然競賽成績不是很好，但是撇掉清大，師大成大我還是有備取到。

那我全國三等獎正取清大的例子我覺得還是看看就好，一方面我這屆許多進過資奧一階的都沒投特殊選材(大概3~到4個)，加上我認為我的面試能力挺好的，我不太會怯場，所以才能苟且正取清大。

結語

特殊選材是一條窄門，但資訊工程系的特殊選才稍微寬了一點點，並沒有想像中的困難，把目標放在清交可能困難一些也很需要運氣，但如果只是中字輩、師大甚至其他學校其實相對簡單很多。

有任何問題都可以私訊我IG、FB，但請一定要註明你是來問我的XD
因為我平常都不接受陌生人的追蹤，所以先私訊我才會理你喔~
